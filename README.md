# Library Inventory Manager

Author: Suhani


Date: 7/12/2025


---

## Description

This is a simple Python program to manage a libraryâ€™s books. It allows you to add new books, search for books by title, issue books to users, return books, and view the entire library inventory. The program saves all data in a catalog.json file so that the library records are preserved between sessions, and it also keeps a library.log file to record all issued and returned books. The interface is command-line based and easy to use.

---

## Key Features 

The program allows users to add new books with title, author, and ISBN.

Users can search for books by their title.

Users can issue and return books.

Users can view all books in the library.

All data is saved automatically in catalog.json.

---

## How to Run

Save the code as main.py.

Run python main.py in the terminal.
